# Tipos de operadores

- Aritméticos
- Atribuição
- Comparação
- Lógica
- Condicional

## Atribuição

operador | exemplo | Equivalente a
---------|---------|--------------
= | x = y | x = y
+= | x += y | x = x + y
-= | x -= y | x = x - y
*= | x *= y | x = x * y
/= | x /= y | x = x / y
%= | x %= y | x = x % y

## Aritmética

operador | Descrição
---------|----------
 + | Adição 
 - | Substração 
 * | Multiplicação 
 ** | Exponenciação 
 / | Divisão 
 % | Módulo 
 ++ | Incrementar
 -- | Decrementar


## Comparação

operador | Descrição
---------|----------
 == | Igual a 
 === | Mesmo valor e mesmo tipo 
 != | Diferente 
 !== | Valor e tipos diferentes 
 > | Maior que 
 < | Menor que 
 >= | Maior ou igual a
 <= | Menor ou igual a

## Lógica

operador | Descrição
---------|----------
 && | "e" lógico
 || | "ou" lógico 
 ! | "não" lógico
