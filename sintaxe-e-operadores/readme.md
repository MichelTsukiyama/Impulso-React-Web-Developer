## Atribuição

operador | exemplo | Equivalente a
---------|---------|--------------
= | x = y | x = y
+= | x += y | x = x + y
-= | x -= y | x = x - y
*= | x *= y | x = x * y
/= | x /= y | x = x / y
%= | x %= y | x = x % y
